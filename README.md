# Calculator
This project is a responsive and fully functional calculator.

#Basic Operations
- Addition
- Subtraction
- Multiplication
- Division

#User Interaction
- Click buttons to input number and operators 
- Use the Keyboard for all numbers and operators
- Press "Enter" to calculate
- Press "Backspace" to delete

#Error Handling
- Displays "Error" when needed

#Design
- Works on mobile, tablet, and desktop
